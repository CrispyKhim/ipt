About

Felis Catus is an interactive web application created by Christopher Khim. The application aims to help users, like yourselves, gain insights into the world of domestic cats. This web application was developed for the HSC IPT Major Project. A multimedia system was to be developed and programmed to be interactive with various media types including texts, images, videos, audio, and animation.

Technology

Felis Catus is built using Python and the Flask framework. The open-source code provided by Bootstrap was utilised and referenced to highlight credibility. This included the implementation of HTML, CSS, and JavaScript code with styling and special effects added.

Other free services / software applications that were used in the making of this system are:

  • Visual Studio Code
  • Obsidian
  • Blender
  • Soundly
  • Audacity
  • GitHub
  • The Cat API
