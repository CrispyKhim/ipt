Copyright

What is Copyright?

To preserve the rights of authors and developers from theft and software piracy, the Copyright Act 1968 (Cwlth) was amended to suit the technological advancements of contemporary society including computer programs, and to protect the work of the author’s and developer’s intellectual property.

Copyright is the right to use, copy, or control the work of authors or copyright holders. It protects the owner’s rights to control how their product is used, however, it only protects the expression or original ideas and not the user of the same idea. Personnels must seek permission of the copyright holder for any material that is used in a product. Graphics, texts, modules, and other media types must not be used unless approved by the copyright holder. If the works are used in a way that violates copyright laws, legal action may be taken.

Our service uses a third-party service in which we call an API. An API, Application Programming Interface, refers to a platform that can be though of as a contract of service between a multitude of applications.

This API is used to retrieve information from the Wikipedia API. The Wikipedia API is an open-source API that allows developers to retrieve information from Wikipedia.

This API is used to retrieve information from the Cat API. The Cat API is an open-source API that allows developers to retrieve information from the Cat API.
